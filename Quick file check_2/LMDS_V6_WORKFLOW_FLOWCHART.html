4 🔍 Search Service — ShipToName-Only Flow

📜 Business Rule v5.4.003: ใช้ ShipToName เท่านั้น ห้ามใช้ ShipToAddress / LatLong_SCG เป็น fallback

⚠️ Business Rule (Single Source of Truth)

ShipToAddress จาก SCG API ไม่น่าเชื่อถือ (อาจมีแค่อำเภอ) → ห้ามใช้เป็น anchor
LatLong_SCG จาก SCG API ยังไม่ verified → ห้าม fallback เขียน LatLong_Actual
AI Reasoning ถูกลบออกใน v5.4.003 → ไม่เหมาะกับ production
